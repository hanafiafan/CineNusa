FilmIndo Setup

Dataset belum ditemukan. Ikuti langkah berikut untuk memulai:

  1. Install Kaggle CLI 
    pip install kaggle
  2. Download dataset 
    kaggle datasets download dionisiusdh/imdb-indonesian-movies
  3. Extract & pindahkan CSV 
    unzip imdb-indonesian-movies.zip -d data/

    Pastikan file CSV berada di folder data/ dalam direktori aplikasi.

  4. Jalankan ulang aplikasi 
    python app.py

    Model SVD akan dilatih otomatis (±1–2 menit pertama kali).

Setelah dataset tersedia, sistem akan secara otomatis:
  • Memuat data film Indonesia
  • Membuat model Content-Based Filtering (TF-IDF)
  • Menghasilkan rating sintetis dan melatih SVD
  • Menyimpan model ke models/svd_model.pkl
